# 📚 编辑指南

这篇文档会详细说明 BNBU 手册的详细编辑要求和规范，供所有参与编辑的同学参考。

## 提交方法

一般而言，所有提交都应通过 Pull Request 合并到 dev 分支上，提交前请先确保您的本地仓库是最新的，以避免不必要的合并冲突。如果遇到冲突，请到编者所在各聊天软件中与相关编者沟通。

## 参考资料格式

一般而言，参考资料并不要求必须提供，但如果您在编辑过程中使用了某些来源的信息，建议您将其添加到文档末尾的参考资料部分，以便其他读者能够验证和进一步了解相关信息。

参考资料一般放于每个 Markdown 文件的末尾，格式如下：

本格式结合了 APA 格式与 Wikipedia 的参考资料格式，适用于网页和书籍两种常见来源。请根据您使用的来源类型选择相应的格式进行引用。

**网页**

```Markdown
(正文)
我校建于2005年[^1]，位于……

(页尾)
## 参考资料

[^1]: **作者姓名/机构名称** (发布年份或具体日期). *文章/页面标题*. 网站或平台名称. 取自: [原始链接](http://example.com) (网页存档: [Internet Archive](http://example.com), 存取于 你访问/存档的日期).

(示例)
[^2]: **宋雪梅** (2020-3-26). *北师港浸大托管会同社区体育公园 不断开拓产学研合作领域*. 珠江晚报. 取自: [原始链接](https://www.bnbu.edu.cn/info/1078/8891.htm) (网页存档: [Internet Archive](https://web.archive.org/web/20260419035308/https://www.bnbu.edu.cn/info/1078/8891.htm), 存取于 2026-04-19).
```

渲染效果：  
我校建于2005年[^1]，位于……  
[^1]: **宋雪梅** (2020-3-26). *北师港浸大托管会同社区体育公园 不断开拓产学研合作领域*. 珠江晚报. 取自: [原始链接](https://www.bnbu.edu.cn/info/1078/8891.htm) (网页存档: [Internet Archive](https://web.archive.org/web/20260419035308/https://www.bnbu.edu.cn/info/1078/8891.htm), 存取于 2026-04-19).

**书籍**

```Markdown
(正文)
我校建于2005年[^3]，位于……

(页尾)
## 参考资料

[^3]: **[作者姓名]** ([出版年份]). *[书籍名称]* ([版本/译者信息，若无则省略]). [出版社名称]. ISBN: [ISBN号码].

(示例)
[^4]: **汤涛** (2023). *UIC与博雅教育*. 广东人民出版社. ISBN: 9787218164571.
```

渲染效果  
我校建于2005年[^2]，位于……
